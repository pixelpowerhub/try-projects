Pixel Power Hub is built for everyone, including people who use screen readers, screen magnifiers, voice control, and keyboard-only navigation. Accessibility isn't a feature we bolt on — it's central to why this site exists.

Our Standard

We aim to meet the Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA. This includes sufficient color contrast, full keyboard operability, clear heading structure, descriptive link text, and compatibility with assistive technology such as NVDA, JAWS, VoiceOver, TalkBack, and Narrator.

What We've Built In

Skip navigation A "Skip to main content" link on every page for keyboard and screen reader users
Semantic structure Proper landmarks, headings, and list markup instead of decorative separators
Keyboard support Visible focus outlines and full operability without a mouse
Talking tools Tools like Math Master include built-in speech feedback and live status regions

Known Limitations

We are continuously auditing and improving the site. Some third-party embedded content (such as advertisements) may not fully meet the same standard. If you encounter any barrier — a missing label, poor contrast, a confusing control, or anything that stops you from using a tool — please tell us.

Report an Accessibility Issue

Please use our Contact Us page to report accessibility barriers. Include the page URL, the assistive technology and browser you were using, and a description of the problem. We treat accessibility reports as a priority.
