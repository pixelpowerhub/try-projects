Pixel Power Hub — User Guide

This guide explains how to use each tool on Pixel Power Hub. All tools work fully in your browser. Your files and data never leave your device.

Password Generator

  1. Choose the length of the password.
  2. Select which character types you want (uppercase, lowercase, numbers, symbols).
  3. Click Generate.
  4. Use Copy to copy the password or Hide to hide it from the screen.

Text Analyzer

  1. Paste text into the text box or upload a .txt file — any language or script works, not just English.
  2. Click Analyse.
  3. View counts for words, characters, lines, paragraphs, and an estimated reading time.
  4. See the most repeated word, and a readability score for English-language text (this score isn't meaningful for non-Latin scripts, so it's hidden for those).

Text Encryptor & Decryptor

  1. Enter or upload your text content.
  2. Select an encryption method (Caesar, ROT13, Base64, etc.) from the dropdown.
  3. Click Encrypt or Decrypt.
  4. Copy or export the result as a text file if needed.

Audio Trimmer

  1. Upload an audio file — most common formats your browser supports will work (MP3, WAV, M4A, OGG, and more).
  2. Use the built-in audio player to preview it and find your start and end points.
  3. Move to the start point and click Set Start Point.
  4. Move to the end point and click Set End Point.
  5. Click Trim Audio, choose WAV or MP3 as your export format, then click Save File.

Document Saver

  1. Enter a desired file name.
  2. Choose the file format (HTML, JSON, CSV, etc.).
  3. Enter your content in the large text area.
  4. Click Save File to download it to your device.

Age Calculator

  1. Select your birth date using the date picker.
  2. Click Calculate.
  3. View your detailed age and countdown to your next birthday.
  4. Use Reset to start over with a new date.

File Encryptor & Decryptor

  1. Choose Encrypt (Lock) or Decrypt (Unlock) using the tabs at the top.
  2. Select your files. You can pick multiple files at once from your device's file picker — tap each one you want, then tap Done. Add more later with Add More Files.
  3. If you select more than one file, they're shown as a collapsed "X files selected" summary. Tap it to expand the list, where you'll see each file's name and a small preview, plus a button to remove it individually — or use Remove All to start over.
  4. Enter a strong password and set your hint questions for extra security (Encrypt), or your original password / hint answers (Decrypt).
  5. Click Lock File & Download or Unlock File to process everything. Files download automatically — use Process Another File to reset the tool.

Privacy Note: Your privacy is our priority. All encryption happens locally in your browser, meaning your files and passwords never leave your device. Because we do not store your data, please keep your password safe, as you are the only one who can access your encrypted files.

Math Master

  1. On your first visit, choose Guest Mode (asks your name each time you revisit) or Create My Profile (remembers your name on this device). You can change your name anytime with the Add Name button at the top.
  2. Select your Category (Addition, BODMAS, Fractions, Squares, and more) and a Level from 1 to 50.
  3. Choose Learning Mode for untimed practice with three tries per question, or Test Mode for a timed challenge.
  4. Pick how many questions you want — 5, 10, 20, 25, 50, 100, or Unlimited Practice — then start your session.
  5. Type your answer and press Enter to submit. Use Hint for a general tip, or Explain for a full step-by-step walkthrough of that exact question, spoken aloud.
  6. Complete a Test Mode session with 70% or higher to unlock and download your Achievement Certificate.

Keyboard shortcuts: Enter submit answer, Ctrl+R repeat question aloud, Ctrl+H hint, Ctrl+K show shortcuts list, Esc exit practice.
