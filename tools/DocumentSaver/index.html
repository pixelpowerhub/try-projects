How to use Document Saver

  1. Type or paste the content you want to save into the text box.
  2. Enter a file name (without an extension — it will be added automatically).
  3. Choose a file format from the dropdown.
  4. Click Save File to download it to your device.

Up to 500,000 characters. JSON and XML content is checked for valid syntax before saving.

Note: Files are generated entirely in your browser and download directly to your device. Nothing you type here is uploaded to or stored on our servers.
