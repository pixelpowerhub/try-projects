How to use Audio Trimmer

  1. Upload an audio file (MP3 or WAV).
  2. Click Play to listen to the audio preview.
  3. Move to the start point and click Set Start.
  4. Move to the end point and click Set End.
  5. Click Trim, then select your format and Save.
